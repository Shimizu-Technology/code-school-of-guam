# Code School of Guam - Internship Program

## Overview

The Code School of Guam internship is an optional 10-week program where graduates work on real production applications in a professional Agile environment. This is a portfolio-building opportunity for graduates who want to gain real-world experience before job hunting.

## Internship Details

- **Duration**: 10 weeks
- **Type**: Optional and unpaid (experience-focused)
- **Environment**: Agile work environment with 2-week sprints
- **Projects**: Real production applications at Shimizu Technology
- **Structure**: Professional software development workflow with daily standups, sprint planning, code reviews, and demos

## What You'll Experience

### Week 1: Onboarding
- Project setup and orientation
- Meet your team and mentors
- Development environment configuration
- Introduction to the codebase

### Weeks 2-10: Sprint-Based Development
- Weekly sprints with assigned tasks
- Sprint planning meetings
- Daily standups
- Sprint retrospectives
- Code reviews
- Real client deliverables

## Skills You'll Develop

- **Technical Skills**
  - Working with production codebases
  - Version control with Git in team settings
  - Code review processes
  - Testing and quality assurance
  - Deployment and DevOps basics

- **Professional Skills**
  - Agile/Scrum methodology
  - Team collaboration
  - Project management
  - Client communication
  - Time estimation

## Paid Opportunities for Top Performers

Separately from the internship, we offer **paid positions** for outstanding graduates:

### 1. Teacher's Assistant (TA) Positions
- **Number of Positions**: 1-2 TAs per cohort
- **Role**: Support the next Code School cohort
- **Benefits**:
  - Paid position
  - Leadership experience
  - Reinforce your own learning
  - Build teaching skills
  - Expand your network

### 2. Junior Software Engineer Contract
- **Position**: Paid contract with Shimizu Technology
- **Role**: Junior Software Engineer
- **Work**: Real client projects
- **Benefits**:
  - Professional experience
  - Competitive compensation
  - Mentorship from senior engineers
  - Portfolio-building projects

These opportunities are merit-based. Show dedication during the bootcamp and internship, and doors will open.

## Internship Projects

You'll work on real applications including:
- Apps for local Guam businesses
- Full-stack web applications
- Projects using Ruby on Rails and React
- AI-powered features and chatbots

Previous cohort projects have addressed Guam-specific challenges, from tourism platforms to local business management tools.

## Eligibility

To participate in the internship:
1. Successfully complete the 20-week bootcamp
2. Demonstrate competency in core skills
3. Maintain good standing (attendance, conduct)
4. Commit to the 10-week internship schedule

Note: The internship is unpaid, but the experience and portfolio projects you gain are invaluable for landing your first tech job.

## Not Just Practice Projects

This is real work for real clients. Your code will be deployed and used by actual businesses. This is what sets Code School of Guam apart from other bootcamps - you graduate with genuine professional experience, not just classroom projects.
