# Code School of Guam - Program Overview

## About the Program

Code School of Guam is Guam's first coding bootcamp, offering a comprehensive 20-week full-stack web development and AI engineering program. The program takes students from complete beginners to job-ready junior developers.

## Program Structure

- **Total Duration**: 22 weeks (5 weeks pre-work + 17 weeks live classes)
- **Format**: Fully remote via Zoom with optional in-person sessions
- **Class Size**: Maximum 10 students per cohort
- **Weekly Commitment**: 20-25 hours per week

## Schedule

- **Live Classes**: Monday through Thursday, 6:00 PM - 9:00 PM (Guam Time/ChST)
- **Weekends**: 3-5 hours of homework and deliberate practice
- **3-Day Weekend**: Friday through Sunday for self-study and projects

## Technologies Taught

### Backend Development
- **Ruby**: Programming language with beginner-friendly, readable syntax
- **Ruby on Rails**: Backend framework for building robust APIs and web applications (used by Airbnb, GitHub, Shopify)
- **PostgreSQL**: Database management

### Frontend Development
- **JavaScript**: Core programming language for web
- **React.js**: Most in-demand frontend framework (used by Meta, Netflix, Airbnb)
- **HTML & CSS**: Web fundamentals

### AI Engineering (NEW)
- **Python**: Programming language for AI/ML
- **OpenAI SDK**: Building AI-powered features and chatbots
- **LLMs & Prompt Engineering**: Working with large language models
- **RAG Systems**: Retrieval-Augmented Generation for chatbots
- **Agentic Systems & Evals**: Building intelligent AI agents

## 20-Week Curriculum Timeline

### Pre-Work Phase (5 Weeks)
- Programming fundamentals using Ruby
- Introduction to HTML, CSS, and JavaScript
- Work with a dedicated pre-work instructor

### Weeks 1-3: Ruby Review & Rails APIs
- Review Ruby fundamentals
- Dive into APIs and Ruby on Rails API development

### Weeks 4-5: JavaScript & React Introduction
- Master JavaScript fundamentals
- Introduction to React for frontend development

### Week 6: GitHub Team Projects
- Collaborative development using Git and GitHub
- Team-based projects

### Week 7: Advanced Rails Development
- Advanced associations and authentication
- Complex data relationships

### Weeks 8-9: Frontend Development Deep Dive
- Advanced React concepts
- State management
- Building dynamic user interfaces

### Weeks 10-13: Python & AI Engineering
- Python fundamentals
- OpenAI SDK & prompt engineering
- Deep dive into LLMs
- Building chatbots
- RAG (Retrieval-Augmented Generation)
- Evals and Agentic Systems

### Weeks 14-15: Capstone Projects
- Design and build full-stack applications
- Rails APIs with React frontends
- AI-powered features
