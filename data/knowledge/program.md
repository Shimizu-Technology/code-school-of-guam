# Code School of Guam - Program Overview

## About the Program

Code School of Guam is Guam's first AI-native coding bootcamp, offering a comprehensive 22-week hybrid full-stack web development and AI engineering program. The program takes students from complete beginners to job-ready junior developers who can build AI-powered applications.

## Program Structure

- **Total Duration**: 22 weeks (5 weeks pre-work + 17 weeks live classes)
- **Format**: Hybrid — live classes via Zoom + structured async practice + individual 1-on-1 mentorship
- **Class Size**: Maximum 10 students per cohort
- **Weekly Commitment**: 13.5+ hours per week (6 hours live + 7.5 hours guided async)

## Hybrid Format — How It Works

### Live Days (Tuesday & Thursday, 6:00–9:00 PM ChST)
- New concepts taught by your instructor via Zoom
- Live coding demos, exercises, and Q&A
- This is where you learn the "what" and "why"

### Async Days (Monday, Wednesday, Friday)
- Structured assignments posted in the learning platform
- Watch assigned lesson recordings (15–30 min each)
- Complete required exercises with auto-grading
- Post completion in Slack — your instructor reviews and gives feedback
- This is where you practice and build muscle memory

### Saturday 1-on-1 Mentorship
- Individual sessions with the Lead Instructor
- Review your progress, work through blockers, plan your week
- 30–60 minutes per student
- No one falls behind — personalized attention every week

**Why hybrid?** Professional developers don't learn everything in a classroom. They read docs, watch tutorials, and build things independently — then ask for help when stuck. Our format trains you to work like a real developer from day one.

## AI-as-a-Tool Philosophy

We don't ban AI — we teach you to use it intelligently with a 6-phase progressive approach:

- **Phase 1 (Weeks 1-3): Build Your Foundation** — No AI tools. Learn fundamentals the hard way, like learning multiplication tables before using a calculator.
- **Phase 2 (Weeks 4-5): AI as Study Buddy** — Introduce Cursor IDE. Use AI to understand concepts, not copy code.
- **Phase 3 (Weeks 6-7): AI as Debugging Partner** — Debug with AI, but understand every fix before applying it.
- **Phase 4 (Weeks 8-9): AI as Coding Accelerator** — Scaffold boilerplate, write tests, speed up development — review every line.
- **Phase 5 (Weeks 10-12): AI Engineering** — Learn how AI works under the hood. Build chatbots, RAG systems, and AI agents.
- **Phase 6 (Weeks 13-17): AI-Native Development** — Build production applications with AI as a core feature.
